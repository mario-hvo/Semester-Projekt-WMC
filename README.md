# Semester-Projekt-WMC

Meine Website ‚Tech & Making – 3D, Drohne, CNC & Coding‘ präsentiert meine Hobby-Projekte aus
den Bereichen 3D-Druck, Drohnen-Fotografie/Videografie, CNC-Fräsen und Programmieren. Sie
enthält Galerie-Seiten mit 3D-Modellen und Drohnenbildern, Erklärungen und Timelapse-Videos zu
CNC-Projekten, eine Sammlung nützlicher Python „learnings“ sowie eine kleine Informatik Fit
Sammlung.
Seite verfügt über ein einheitliches Design mit externem CSS, Responsive-Elementen, einem
Kontaktformular mit JavaScript-Validierung.
